# Sentiment Analysis 
Built using Flask and Python it evaluates the emotional tone of text. The app provides real-time classification (Positive, Negative, or Neutral) along with precise polarity and subjectivity metrics.

Polarity is used to measure the emotional direction of the text and  ranges from -1 (negative) to 1 (positive) values.

Subjectivity shows the degree to which the text is influenced by the author's own personal bias, opinions and beliefs with it's value ranging from 0 (objective) to 1 (subjective).

Tech Stack
Backend: Python, Flask

NLP Engine: TextBlob

Frontend: HTML5, CSS3
